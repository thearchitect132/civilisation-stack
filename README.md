Civilisation‑Stack
A framework for modelling, designing, and architecting civilisation‑scale cognitive systems.

The Civilisation‑Stack is an open, evolving architecture that formalises how a civilisation thinks, remembers, coordinates, and acts across time. It is a blueprint for building Tammetry — a civilisation‑scale cognitive architecture — and for defining TRU, the unit of symbolic participation within shared capability systems.

This repository contains the conceptual foundations, whitepapers, architecture layers, research notes, and prototype models that form the early structure of the Civilisation‑Stack.

Purpose
The Civilisation‑Stack exists to answer a single question:

How does a civilisation become a coherent cognitive agent?

To do this, the stack provides:

A layered architecture for modelling collective cognition

A symbolic participation ledger (TRU)

A mythic‑technical narrative framework

A research corpus for long‑horizon system design

A whitepaper series documenting the evolving architecture

Core Concepts
1. Tammetry
Tammetry is the civilisation‑scale cognitive architecture that coordinates distributed intelligence across time, agents, and symbolic systems.
Explore more: Tammetry

2. TRU — The Unit of Participation
TRU is the symbolic ledger entry that encodes contribution, alignment, and participation within Tammetric systems.
Explore more: TRU

3. The Four Layers of the Civilisation‑Stack
Core Layer — primitives, operators, identity, cognition

Mid Layer — coordination, protocols, symbolic exchange

Deep Layer — mythic‑technical narrative, civilisation identity

Envelope Layer — interfaces, publication, external coherence
